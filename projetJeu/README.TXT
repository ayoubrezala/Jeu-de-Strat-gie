		========================
		COMPILATION & EXECUTION
		========================
Pour compiler le projet en ligne de commande:
placez vous dans la racine du projet et en suite taper les commandes suivante:
$ ant compile : compilation
$ ant run : execution

Pour exécuter le projet depuis la ligne de commande, accédez au dossier dist et
tapez la commande suivante:
$ java -jar "projetJeuConsole.jar" pour executer la partie console

$ java -jar "projetJeuGraphic.jar" pour executer la partie graphique
